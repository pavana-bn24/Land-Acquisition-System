# Land Acquisition System

The Land Acquisition System is a web-based application developed to manage land plots, user registrations, and reservations in an organized and efficient manner. The system provides a platform for users to explore available plots and reserve them, while administrators can manage plot details, monitor reservations, and generate reports.

## Project Overview

This project was developed as a mini project during my internship at Swizosoft OPC Private Limited. The main objective of the system is to replace manual land management processes with a digital solution that ensures accuracy, transparency, and ease of access.

The application demonstrates the implementation of a full-stack web system where the frontend interacts with the backend database to perform real-time operations.

## Features

- User registration and authentication  
- Secure login system with password validation  
- Display of available plots  
- Plot reservation functionality  
- Automatic update of plot status  
- Admin dashboard for managing plots  
- Update plot status (Available, Reserved, Sold)  
- Report generation  
- Print functionality  

## System Modules

### User Module
Allows users to register, log in, view available plots, and reserve plots.

### Admin Module
Provides administrative control to manage plots, update status, and monitor reservations.

### Report Module
Generates structured reports using database data.
